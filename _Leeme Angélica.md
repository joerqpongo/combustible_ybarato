## Para ejecutar el programa:

Para poner en marcha la aplicación web en cualquier ordenador, sigue estos sencillos pasos:

### ⚙️ Requisitos previos
Asegúrate de tener instalado **Python 3.9 o superior** en el sistema. Puedes descargarlo desde [python.org](https://www.python.org/downloads/).

### 💻 Pasos para la ejecución

#### 1. Instalar las dependencias
Abre una terminal o consola de comandos en la carpeta de este proyecto y ejecuta el siguiente comando para instalar las librerías necesarias:
```bash
pip install -r requirements.txt
```

#### 2. Arrancar la aplicación
Tienes dos opciones para iniciar el programa:

* **Opción A (Rápida en Windows):** 
  Haz doble clic directamente sobre el archivo **`Iniciar_App.bat`** que se encuentra en la carpeta raíz del proyecto.
  
* **Opción B (Cualquier sistema operativo - Windows, Mac o Linux):** 
  Escribe el siguiente comando en la terminal y presiona Enter:
  ```bash
  python -m streamlit run app.py
  ```

#### 3. Acceder al navegador
Una vez ejecutado, la aplicación se abrirá de forma automática en tu navegador. Si no lo hace, puedes entrar escribiendo la siguiente dirección en la barra de navegación:
👉 **[http://localhost:8501](http://localhost:8501)**
