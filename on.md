# Instrucciones para arrancar el programa

Para poner en funcionamiento la aplicación web, sigue estos pasos:

1. Abre una terminal (o consola de comandos) en esta misma carpeta (`combustible_ybarato.es`).
2. Ejecuta el siguiente comando:

```bash
python -m streamlit run app.py
```

*(Nota: Si tienes la ruta de los scripts de Python configurada correctamente, también puedes usar simplemente `streamlit run app.py`)*

3. La aplicación se iniciará y automáticamente se abrirá una nueva pestaña en tu navegador web predeterminado (por lo general, en la dirección `http://localhost:8501`).
