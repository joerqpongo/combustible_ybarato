# ⛽ Cómo iniciar la aplicación combustible_ybarato.es

Esta es una guía rápida para poner en marcha la aplicación web que busca las gasolineras más baratas en España.

---

## 🛠️ Requisitos previos

Asegúrate de tener instalado:
* **Python 3.9** o superior.
* Las dependencias del proyecto instaladas.

Si aún no las has instalado, abre una terminal en esta carpeta y ejecuta:
```bash
pip install -r requirements.txt
```

---

## 🚀 Inicio rápido (Local)

Para iniciar la aplicación localmente en modo de desarrollo:

### Opción A (Recomendada)
Ejecuta el siguiente comando en la terminal:
```bash
python -m streamlit run app.py
```

### Opción B
Si tienes las rutas de scripts configuradas en tu sistema, también puedes usar:
```bash
streamlit run app.py
```

---

## 💻 Acceso a la aplicación

Una vez iniciado el servidor, la aplicación abrirá automáticamente una pestaña en tu navegador web predeterminado. Si no se abre, puedes acceder de forma manual escribiendo la siguiente dirección en tu navegador:

🔗 **[http://localhost:8501](http://localhost:8501)**

---

## 🛑 Cómo detener la aplicación

Para apagar el servidor local:
1. Ve a la consola/terminal donde se está ejecutando el comando de inicio.
2. Presiona la combinación de teclas **`Ctrl + C`**.
3. Cierra la pestaña en tu navegador.
