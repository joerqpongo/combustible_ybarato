# Instrucciones para apagar el programa

Si necesitas detener la aplicación web, sigue estos sencillos pasos:

1. Ve a la ventana de la terminal (o consola de comandos) donde iniciaste el programa y donde se está ejecutando actualmente.
2. Presiona la siguiente combinación de teclas:

   **`Ctrl + C`**

3. Esto enviará una señal de interrupción al servidor local de Streamlit, deteniendo el proceso de forma segura y liberando el puerto.
4. Una vez hecho esto, puedes cerrar la pestaña de la aplicación en tu navegador web (si intentas navegar o interactuar con ella después de detener el servidor, mostrará un error de conexión, lo cual es normal).
